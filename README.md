# Vehicle Automation App

An Android application built during our Diploma as a team project with 3 friends. The app connects vehicle owners with garages/service centers — users can browse garages, book services, track order history, leave feedback/ratings, and chat for support, while admins manage garages and listings.

## Features

- User & admin authentication (login/register)
- Garage listing and management (add/update garage/profile)
- Service booking with order history tracking
- Ratings & feedback system
- In-app chatbot support
- Location-based garage discovery
- Payment integration
- User profile management

## Tech Stack

- **Language:** Kotlin & Java
- **Platform:** Android (Gradle build)
- **Backend:** Firebase

## Getting Started

1. Clone the repo and open it in Android Studio.
2. Create your own Firebase project and download `google-services.json`, placing it in `app/`. (This file is excluded from version control since it contains project-specific keys.)
3. Sync Gradle and run the app on an emulator or device.
